Gantt v3 · interactions layer

Opt-in interactions on top of the existing PDF sheet.

Il componente base GanttSheet resta "sola resa". Tutte le nuove capacità sono opzionali via props: se il consumer non passa callback, nessun layer attivo, zero regressione per AutoBuilderGantt e /gantt-preview. Copre il 100% delle interazioni di PdcGanttV2 (drag move / resize / cross-day / action bar 8-icone / timeline click / hover tooltip / autoFit).

S1 · idle

Stato idle — identico a oggi

consumer senza callback → comportamento GanttSheet attuale
ganttId=undefined · autoFit=false
Render puroniente overlay, niente handle, niente cursori particolari. Il cursor è default ovunque.
Tooltip nativohover sul segmento → <title> SVG (browser), come oggi. Tooltip custom JS solo se il consumer opta-in.
Esempi d'usoAutoBuilderGantt, /gantt-preview. Nessuna prop nuova richiesta.
S2 · selected

Selected state + action bar 8 icone

single-click su segmento → ring kinetic + bar sopra
onAction + hideActionBar=false
Ordine icone (stesso di oggi)✎ ↔ ⧉ | 🔗 ⚠ | ↗ ⧗ | × · 3 separatori verticali · warn in ambra, delete in rosso.
Clamp ai bordise il centro blocco è a x<150, la bar si sposta a sinistra ma la freccia ::after resta centrata sul blocco via --arrow-x.
Dismissclick fuori dall'SVG o Esc → deseleziona + fade out action bar. hideActionBar=true disabilita l'overlay (PdcPage "lettura").
S3 · drag

Drag in progress — ghost + slot indicator + sticky time

threshold 4px · snap 5min · CVp/CVa seguono il treno padrone
onSegmentDrag attivo · cursor grabbing
11:35 → 13:51
Ghostbordo tratteggiato brand sul rettangolo in drag (niente copia OS-style). Posizione originale mantenuta sfumata al 8% opacity come "back-reference".
Slot sull'asseriga verticale 2px in brand #0062CC con pill orario in alto (::before). Snap visivo ogni 5min.
Sticky timetooltip dark ancorato al cursore, sempre visibile durante resize/drag. Mostra dep → arr aggiornato real-time.
S4 · cross-day

Cross-day drag — source → target con drop-slot

HTML5 DnD · MIME application/x-colazione-block · ganttId obbligatorio
source · ganttId=g1-LMXGV · blocco in uscita
target · ganttId=g3-D · drop-slot attivo
MIME payload{ganttId, block, index, linkedCvp?, linkedCva?} su application/x-colazione-block. Fallback text/plain per browser strict.
Flow atomicosource chiama onCrossDragStart → target chiama onCrossDrop(payload, targetId, time) → source chiama onCrossRemove(idx, withLinkedCvs) se dropEffect==="move".
Drop rejectionse payload.ganttId === target.ganttId il drop viene ignorato (il drag intra-Gantt gestisce già move temporale).

API estesa · opt-in completa

Props aggiunte a GanttSheetProps

// Identity (obbligatorio solo per cross-day DnD)
ganttId?: string

// Drag temporale intra-Gantt (move + resize)
onSegmentDrag?: (
  rowIdx: number,
  segIdx: number,
  changes: { dep_time?: string; arr_time?: string },
) => void

// Click su area asse vuota (add-block in Builder)
onTimelineClick?: (
  hour: number,
  minute: number,
  rowIdx: number,
) => void

// Cross-day DnD
onCrossDragStart?: (p: CrossDragPayload) => void
onCrossDrop?: (
  p: CrossDragPayload,
  targetGanttId: string,
  dropTime: { hour: number; minute: number; rowIdx: number },
) => void
onCrossRemove?: (
  segIdx: number,
  withLinkedCvs: boolean,
) => void

// Action bar 8-icone
onAction?: (
  action: GanttAction,
  seg: GanttSegment,
  rowIdx: number,
  segIdx: number,
) => void

// Tuning
hideActionBar?: boolean    // default false
autoFit?: boolean          // default false
snapMinutes?: number       // default 5
dragThresholdPx?: number   // default 4
